# Docker 快速体验

不想装 Python 和 PostgreSQL？用 Docker 三条命令即可体验 Riverse。

## 快速开始

访问 **[Riverse-Docker](https://github.com/wangjiake/Riverse-Docker)** 仓库获取完整说明。

支持：

- OpenAI
- DeepSeek
- Groq
- 任何 OpenAI 兼容接口

## 包含内容

Docker 部署包含：

- Python 运行时
- PostgreSQL 数据库
- Riverse 应用
- 所有依赖

无需手动安装 Python、PostgreSQL 或 Ollama — 只需提供 API Key 即可开始对话。
