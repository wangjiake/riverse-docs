# Docker Quick Start

Don't want to install Python or PostgreSQL? Run Riverse with Docker in 3 commands.

## Quick Start

Visit the **[Docker guide](https://github.com/wangjiake/JKRiver/tree/main/docker)** in the JKRiver repository for full instructions.

Supports:

- OpenAI
- DeepSeek
- Groq
- Any OpenAI-compatible API

## What's Included

The Docker setup bundles:

- Python runtime
- PostgreSQL database
- Riverse application
- All dependencies

No manual installation of Python, PostgreSQL, or Ollama needed — just provide your API key and start chatting.
