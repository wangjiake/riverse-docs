# Docker クイックスタート

Python や PostgreSQL をインストールしたくない方へ。Docker で 3 コマンドで Riverse を体験できます。

## クイックスタート

**[Riverse-Docker](https://github.com/wangjiake/Riverse-Docker)** リポジトリで完全な手順をご確認ください。

対応：

- OpenAI
- DeepSeek
- Groq
- その他の OpenAI 互換 API

## 含まれるもの

Docker セットアップに含まれるもの：

- Python ランタイム
- PostgreSQL データベース
- Riverse アプリケーション
- すべての依存パッケージ

Python、PostgreSQL、Ollama の手動インストール不要 — API キーを入力するだけですぐに会話開始。
