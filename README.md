# Riverse Docs

The official documentation site for [Riverse](https://github.com/wangjiake/JKRiver) — a personal AI agent with persistent memory powered by the River Algorithm.

**Live site: [wangjiake.github.io/riverse-docs](https://wangjiake.github.io/riverse-docs/)**

## Languages

- English
- 中文
- 日本語

## Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Serve locally
mkdocs serve

# Build
mkdocs build

# Deploy to GitHub Pages
mkdocs gh-deploy --force
```

## Related Projects

- [JKRiver](https://github.com/wangjiake/JKRiver) — Main agent (source code)
